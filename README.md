

Este proyecto es un panel interactivo de análisis de estudiantes universitarios desarrollado con Streamlit, Pandas y Matplotlib/Seaborn.  
Permite visualizar indicadores clave como tasas de retención, niveles de satisfacción y matrículas por año, departamento y término académico.

---


El University Dashboard ofrece una interfaz sencilla y dinámica para analizar el rendimiento y la distribución de estudiantes en diferentes programas universitarios.  
Los datos provienen del archivo `university_student_data.csv`, que contiene información sobre:

- Año académico (`Year`)
- Departamento (`Department`)
- Término o semestre (`Term`)
- Tasa de retención (`Retention_Rate`)
- Puntaje de satisfacción (`Satisfaction_Score`)
- Total de matrículas (`Enrollments`)
